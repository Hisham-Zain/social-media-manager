# ⌨️ Keyboard Shortcuts

Quick reference for AgencyOS keyboard shortcuts.

---

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New project |
| `Ctrl+O` | Open file |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+Q` | Quit |
| `F11` | Toggle fullscreen |
| `Escape` | Close dialog |

---

## Navigation

| Shortcut | View |
|----------|------|
| `Ctrl+1` | Dashboard |
| `Ctrl+2` | Content Studio |
| `Ctrl+3` | Media Library |
| `Ctrl+4` | Automation |
| `Ctrl+5` | Strategy Room |
| `Ctrl+6` | AI Tools |
| `Ctrl+7` | Job Queue |
| `Ctrl+8` | Settings |

---

## Content Studio

| Shortcut | Action |
|----------|--------|
| `Ctrl+G` | Generate (AI) |
| `Ctrl+P` | Preview |
| `Ctrl+E` | Export |
| `Ctrl+Enter` | Run current action |
| `Space` | Play/Pause preview |

---

## Timeline

| Shortcut | Action |
|----------|--------|
| `←` / `→` | Move playhead |
| `Home` / `End` | Go to start/end |
| `Delete` | Remove selected clip |
| `Ctrl+C` | Copy clip |
| `Ctrl+V` | Paste clip |
| `Ctrl+D` | Duplicate clip |

---

## Media Library

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Search |
| `Delete` | Delete selected |
| `Enter` | Open/preview |
| `Ctrl+A` | Select all |

---

## Job Queue

| Shortcut | Action |
|----------|--------|
| `R` | Retry failed job |
| `Delete` | Cancel job |
| `↑` / `↓` | Change priority |
| `Ctrl+Shift+C` | Clear completed |

---

## AI Tools

| Shortcut | Action |
|----------|--------|
| `Ctrl+Enter` | Execute tool |
| `Ctrl+Shift+G` | Generate with AI |
| `Tab` | Next field |
| `Shift+Tab` | Previous field |

---

## Custom Shortcuts

Configure custom shortcuts in **⚙️ Settings** → **Keyboard**.
